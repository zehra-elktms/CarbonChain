
# CarbonChain 
## TÜBİTAK 2209-A Destekli Araştırma Projesi

CarbonChain, şirketlerin karbon ayak izi verilerini güvenli, şeffaf ve doğrulanabilir şekilde yönetmesini sağlayan Hyperledger Fabric tabanlı bir karbon yönetim platformudur.

### Proje Amacı

Bu proje ile:

- Karbon emisyon kayıtlarının güvenli tutulması
- Karbon verilerinin doğrulanması
- Token tabanlı teşvik mekanizması oluşturulması
- Kurumsal sürdürülebilirliğin desteklenmesi
- Karbon ticaret süreçlerinin dijitalleştirilmesi

amaçlanmaktadır.
